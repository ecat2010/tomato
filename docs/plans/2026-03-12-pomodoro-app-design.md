# 番茄時鐘高質感單頁應用設計

日期：2026-03-12

## 目標

建立一個可直接運行、純前端嘅番茄時鐘網頁應用，支援專注／短休息／長休息三種模式切換、自動階段流轉、本地保存設定與當日完成次數，同時以高級霧面系視覺語言呈現，避免做成普通練習 demo。

## 產品方向

成品會係一個單頁、單主卡片嘅沉浸式工具。版面核心只圍繞一個任務：幫用戶進入節奏、維持專注、平滑切換休息階段。畫面採用深灰、煙粉、暗紅為主色，搭配柔和漸變背景、玻璃擬態卡片、霧化光斑與克制動畫，令工具感覺更似精緻產品，而唔係簡單計時器。

## 資訊架構

### 主視覺區

- 頂部顯示品牌式標題與一句短鼓勵文案
- 中央用大型環形進度條包住倒數時間
- 倒數數字使用 tabular figures，避免數字跳動造成版面抖動
- 顯示當前模式名稱，例如「專注中」、「短休息」、「長休息」

### 操作區

- 模式切換 segmented control：專注／短休息／長休息
- 主操作按鈕：開始、暫停、重置、跳到下一階段
- 按鈕提供 hover、active、disabled 狀態與細微縮放

### 狀態與設定區

- 顯示當前番茄輪次
- 顯示今日完成次數
- 內嵌設定面板，可修改專注／短休息／長休息時長
- 額外提供深色模式切換

## 互動與狀態流

### 計時規則

- 預設專注 25 分鐘、短休息 5 分鐘、長休息 15 分鐘
- 開始後每秒更新剩餘時間
- 暫停時凍結計時，恢復時繼續，不可重複建立 interval
- 重置會回到目前模式嘅完整時長
- 手動切換模式時，如正在計時，會重新載入該模式時長並停止播放中狀態
- 「跳到下一階段」會按番茄循環規則切換去下一個模式

### 自動流轉

- 每次階段完成會播放一段輕提示音
- 同步觸發柔和發光／脈衝動畫，同時顯示一條短狀態提示
- 專注完成時今日完成次數加一，並更新番茄輪次
- 系統會自動切去下一階段並準備下一輪時間
- 長休息觸發規則採用「每完成 4 次專注後進入長休息」

### 文件標題

- 計時中標題格式例如 `24:31 - 專注中`
- 暫停或閒置時保留模式相關標題，例如 `已暫停 - 專注`

## 視覺與動效設計

### 視覺語言

- 背景：多層深灰到暗紅漸變，加入低透明模糊光斑
- 主卡：半透明深色玻璃卡，帶細緻邊框、高斯模糊與柔和陰影
- 色彩：主行動用珊瑚暗紅，次要資訊用暖灰與煙粉
- 字體：優先使用系統可得但質感較高嘅字體組合，標題偏優雅，數字偏穩定

### 動畫節奏

- 模式切換採用淡入淡出同輕微位移過渡
- 進度環用 stroke-dashoffset 平滑更新
- 完成狀態會令主卡外圍光暈短暫增強
- 遵守 reduced motion，減少非必要動畫

## 技術方案

- `Vite + React + TypeScript`
- `Tailwind CSS` 做設計 token、排版與狀態樣式
- React hooks 管理 timer 狀態、localStorage 同 document title
- 使用 SVG 建立環形進度條
- 使用 Web Audio API 產生輕提示音，避免額外音檔依賴

## 本地保存策略

- `localStorage` 保存：
  - 專注／短休息／長休息時長
  - 深色模式偏好
  - 今日完成次數
  - 當日日期標記
- 如日期已變更，自動重設今日完成次數

## 元件拆分

- `App`：頁面組裝與全域狀態連接
- `PomodoroCard`：主卡容器與區塊佈局
- `ModeTabs`：三模式切換
- `TimerRing`：環形進度與大倒數
- `ControlBar`：開始／暫停／重置／下一階段
- `StatsPanel`：輪次與今日完成數
- `SettingsPanel`：時長設定與深色模式
- `usePomodoroTimer`：核心計時與流轉邏輯
- `useLocalStorage`：本地資料保存讀寫

## 測試重點

- 計時開始／暫停／恢復唔會產生重複 interval
- 模式切換與重置時間正確
- 專注完成後會更新今日完成次數與下一階段
- 第 4 次專注完成後切換到長休息
- localStorage 讀寫與跨日重設正確
- document title 會隨狀態更新

## 風險與對策

- 瀏覽器背景分頁可能導致 setInterval 漂移：以 timestamp 差值計算剩餘時間，避免累積誤差
- 音訊播放可能受瀏覽器互動限制：首次用戶互動後初始化 AudioContext
- 手機版高度不足：採用單卡片垂直堆疊與 `min-h-dvh`

## 成功標準

- 直接 `npm install` / `npm run dev` 可啟動
- 桌面與手機版都保持高質感與易用性
- 計時邏輯穩定，切換流程自然
- UI 完整、有產品感、唔似普通 demo
